## 第三步：部署 Live Demo（让面试官在线体验）

### 方案 A: Railway（推荐，最简单的免费部署）

1. 在项目根目录创建 `Procfile`:
```
web: uvicorn backend.main:app --host 0.0.0.0 --port $PORT
```

2. 在 `railway.app` 注册 → New Project → Deploy from GitHub Repo
3. 选择 `ai-short-drama-platform` 仓库
4. 设置环境变量: `DEEPSEEK_API_KEY=sk-xxx`
5. 自动部署 → 获得访问 URL

**优点**: 免费额度够用、自动 HTTPS、支持环境变量
**缺点**: 免费版在30分钟无访问后会休眠

### 方案 B: Vercel（如果你是前端项目，不太适合 Python）

你的项目是 Python FastAPI，Vercel 对 Python 支持一般。

### 方案 C: Hugging Face Spaces（完全免费，适合展示）

1. 创建 `Dockerfile`:
```dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
CMD ["uvicorn", "backend.main:app", "--host", "0.0.0.0", "--port", "7860"]
```

2. 在 huggingface.co → New Space → 选择 Docker
3. 关联 GitHub 仓库 → 自动构建

**优点**: 完全免费、无休眠、支持 Docker
**适合**: 面试 Demo 展示
