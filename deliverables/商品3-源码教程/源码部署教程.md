## 二、环境配置（3 步跑通）

### 第 1 步：安装 Python

下载 Python 3.10+ → https://python.org → 安装时勾选"Add Python to PATH"

验证安装成功：

```bash
python --version
# 输出: Python 3.10.11
```

### 第 2 步：安装依赖

打开终端，进入项目目录：

```bash
cd ai-short-drama-platform
pip install -r requirements.txt
```

### 第 3 步：运行 Demo（不需要 API Key）

```bash
# 打开在线 Demo（浏览器直接打开，无需任何安装）
start docs/index.html

# 或命令行运行
python -m scripts.cli demo                    # 短剧场景
python -m scripts.cli demo --scene product_showcase   # 电商场景
python -m scripts.cli demo --scene knowledge_short     # 知识口播
python -m scripts.cli demo --scene cross_border        # 跨境场景
```

### 面试必问的 3 个问题 + 回答

**Q1：Agent 怎么设计的？**

> 我用了 React 模式。每个 Agent 有 think（意图识别）→ act（执行）→ observe（自检）→ respond（输出）四个阶段。7 个 Agent 由 Director 统一调度，Analyst 的 3 个子任务并行执行，Storyboarder 汇总后交给 PromptEngineer 并行生成图片和视频 Prompt，最后 Reviewer 做质量审核。

**Q2：安全怎么做的？**

> 三层防御。第一层是消息结构隔离，用 safe_chat 把用户指令和用户数据分到两个不同的 message 里，防止提示词注入。第二层是 ContentSafetyScanner，在文本到 LLM 之前做 7 个阶段的内容安全扫描，覆盖时间桥、委婉语、对抗扰动等 12 类攻击。第三层是输出完整性校验，防止被注入的质量评分欺骗。

**Q3：怎么证明有效性？**

> 我设计了 88 个边界测试探测，覆盖 9 个维度，全部通过。在线 Demo 可以直接体验，GitHub 有完整的安全审计报告。
