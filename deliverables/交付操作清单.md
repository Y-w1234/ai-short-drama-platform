# 闲鱼交付操作清单

## 收到订单后 3 步完成交付

---

### 商品 1：短剧分镜方案（¥29）

**客户拍下后：**

1. 打开闲鱼聊天框，回复：
   > 收到！请把剧本发给我，200-2000 字最佳。
   > 收到后 24 小时内交付完整方案。

2. 收到剧本后，在终端运行：
   ```bash
   cd D:/claude/ai-short-drama-platform
   python -m scripts.demo --scene short_drama
   ```
   （目前是内置剧本，将来接入真实 API 后替换为客户的剧本）

3. 输出结果在 `scripts/output/demo_short_drama.json`

4. 用这个 JSON 填充模板，生成交付文件。

5. 参考商品 `商品1-短剧分镜方案/求婚-完整分镜方案.md` 的格式。

6. 导出为 PDF（用 Typora 或浏览器打印功能），发给客户。

7. 在闲鱼点「已发货」。

---

### 商品 2：带货视频方案（¥19）

**客户拍下后：**

1. 打开闲鱼聊天框，回复：
   > 收到！请发送以下信息：
   > ① 商品名称
   > ② 3-5 个核心卖点
   > ③ 目标人群
   > ④ 价格
   > 收到后 24 小时内交付。

2. 收到信息后，参考 `商品2-带货视频方案/蓝牙耳机-带货视频方案.md` 的格式。

3. 把客户的商品信息填进去，调整卖点拆解和分镜。

4. 交付物：MD 文件 + PDF 版（含分镜脚本 + Prompt + 口播文案）

5. 在闲鱼点「已发货」。

---

## 售后话术

### 交付时

```
方案已出，请查收文件 📎

文件说明：
- .md 文件：方案全文（推荐用 Typora 打开）
- .pdf 文件：阅读版（手机直接看）

Prompt 可以直接复制到即梦/Midjourney 使用。
有任何问题随时找我，3 天内免费修改。
```

### 客户满意后

```
感谢支持！如果可以的话，麻烦给个好评 🙏
这对我的小店帮助很大！

（好评后截图发我，下次下单 8 折）
```

### 客户不满意

```
收到反馈！我帮你改这几点：
（列出要改的地方）

修改后重新发送，改完还不满意可以退款。
```

## 退款处理

- 修改 2 次后仍不满意 → 全额退款（闲鱼虚拟商品本身不支持强制退款，这句话是为了信誉）
- 客户拍下后 24 小时内未提供素材 → 主动退款（避免差评）
